## ğŸš€ Getting Started

### 1. Browse the Lessons
All lessons are written in **Markdown** (`.md`) format for easy reading on GitHub or any text editor. Navigate to the file corresponding to your interest (e.g., `python_lesson_01.md`).

### 2. Convert to HTML (Optional)
This repository includes a utility script to convert the Markdown lessons into HTML files for a better reading experience in your browser.

**Prerequisites:**
- Python 3.x installed on your system.
- `markdown` library installed (`pip install markdown`).

**Usage:**
1.  Open your terminal in the project directory.
2.  Run the script:
    ```bash
    python markdown2html.py
    ```
3.  Enter the filename when prompted (e.g., `python_lesson_01.md`).
4.  Open the generated `.html` file in your favorite browser.
