# Official Documentation and Learning Resources

This document contains a curated list of official documentation and high-quality learning resources for the programming languages and technologies covered in the syllabus.

## C Programming
*   **Official Standard:** [ISO C Standard (ISO/IEC 9899)](https://www.iso.org/standard/74528.html) - *Note: The official standard is a paid document, but drafts are often available.*
*   **Reference:** [cppreference.com (C)](https://en.cppreference.com/w/c) - Widely used community-maintained reference.
*   **Microsoft Docs:** [C Language Reference](https://learn.microsoft.com/en-us/cpp/c-language/) - For MSVC implementation.
*   **GNU Manual:** [GNU C Reference Manual](https://www.gnu.org/software/gnu-c-manual/) - For GCC implementation.

## Assembly Language (x86, ARM)
*   **x86 Official:** [Intel Software Developer's Manuals](https://www.intel.com/content/www/us/en/developer/articles/technical/intel-sdm.html) - The definitive reference for x86/x64.
*   **ARM Official:** [ARM Architecture Documentation](https://developer.arm.com/documentation) - Official guides for AArch64 and ARM32.
*   **Learning (x86):** [Guide to x86 Assembly (University of Virginia)](https://www.cs.virginia.edu/~evans/cs216/guides/x86.html)
*   **Learning (ARM):** [Azeria Labs](https://azeria-labs.com/) - Focuses on ARM assembly and security.

## C++ Programming
*   **Reference:** [cppreference.com (C++)](https://en.cppreference.com/w/cpp) - The gold standard for online C++ documentation.
*   **Standard Committee:** [isocpp.org](https://isocpp.org/) - News and status of the C++ standard.
*   **Microsoft Docs:** [C++ Language Reference](https://learn.microsoft.com/en-us/cpp/cpp/)
*   **Tutorial:** [learncpp.com](https://www.learncpp.com/) - Highly recommended, up-to-date tutorial series.
*   **Best Practices:** [C++ Core Guidelines](https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines) - Edited by Bjarne Stroustrup and Herb Sutter.

## JavaScript Programming
*   **Official Documentation:** [MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript) - The most comprehensive resource for JS and Web APIs.
*   **Specification:** [ECMAScript Language Specification](https://tc39.es/ecma262/)
*   **Tutorial:** [JavaScript.info](https://javascript.info/) - Modern, detailed tutorial.
*   **Node.js:** [Node.js Documentation](https://nodejs.org/en/docs/)
*   **Vue.js:** [Vue.js Documentation](https://vuejs.org/guide/introduction.html)
